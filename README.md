# Execute in dev environment

1. Clone repository
2. Execute
```
npm install
```
3. Install Nest CLI
```
npm i -g @nestjs/cli
```
4. Run database 
```
docker-compose up -d
```

5. Seed data
```
http://localhost:3000/api/v2/seed
```

## Stack
* MongoDB
* Nest
